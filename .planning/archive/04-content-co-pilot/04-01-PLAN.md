**1. lib/routers/copilot.js** â€” Create tRPC router with protectedProcedure:

`getRecentThread` query:
- Find first CopilotThread for ctx.user.id, orderBy updatedAt desc
- Include messages ordered by createdAt asc
- Return thread or null

`createThread` mutation:
- Input: { accountId: z.string().optional() }
- Create CopilotThread with userId from ctx.user.id and optional accountId
- Return the new thread

`getDailyUsage` query:
- Count CopilotMessage with role "user" for this user where createdAt >= today midnight
- Return { count: number, limit: 50, warning: boolean (count >= 50) }

`getSuggestionChips` query:
- Read latest AIInsight entries for PERFORMANCE_PATTERN, COMPETITOR_STRATEGY, AUDIENCE_QUESTION (same pattern as intel-context.js)
- Generate 3-4 suggestion strings from the data, e.g.:
  - From performance: "Write a thread about [top topic from patterns]"
  - From competitors: "Counter [competitor theme] with our take"
  - From audience: "Answer: [top unanswered question]"
  - Fallback: "Draft a post about [general topic]"
- Return array of { id, label, prompt } objects
- If no intel data, return generic suggestions like "Draft a thread about our latest update", "What topics are trending?"
