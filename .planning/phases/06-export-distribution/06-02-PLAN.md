   - Create nodemailer transport at module scope (for warm reuse in serverless):
     ```javascript
     const transport = nodemailer.createTransport({
       host: process.env.SMTP_HOST,
       port: parseInt(process.env.SMTP_PORT || '587'),
       secure: process.env.SMTP_PORT === '465',
       auth: { user: process.env.SMTP_USER, pass: process.env.SMTP_PASS },
     });
     ```
   - Export `sendReportEmail({ report, recipients, pdfBuffer, appUrl })`:
     a. Import render from @react-email/render, ReportEmail from email-templates
     b. Render email HTML: `await render(<ReportEmail report={report} appUrl={appUrl} />)`
     c. Build attachments array: if pdfBuffer, add `{ filename: sanitized-title.pdf, content: pdfBuffer, contentType: 'application/pdf' }`
     d. Call `transport.sendMail({ from: process.env.SMTP_FROM || 'Social Command <noreply@figure.com>', to: recipients.join(', '), subject: \`Report: \${report.title}\`, html, attachments })`
     e. Return `{ messageId: info.messageId, accepted: info.accepted, rejected: info.rejected }`

3. Extend `lib/routers/reports.js` with two new procedures:

   a. `emailReport` mutation:
      - Input: `z.object({ reportId: z.string(), recipients: z.array(z.string().email()).min(1), subject: z.string().optional() })`
      - Fetch report by ID (throw if not found)
      - Generate PDF buffer via `renderReportPDF(report)` from pdf-renderer (for attachment)
      - Determine appUrl from `process.env.NEXTAUTH_URL || 'https://app.socialcommand.com'`
      - Call `sendReportEmail({ report, recipients: input.recipients, pdfBuffer, appUrl })`
      - On success: create ReportDelivery with channel 'EMAIL', recipients JSON, status 'SENT'
      - On failure: create ReportDelivery with channel 'EMAIL', recipients JSON, status 'FAILED', error message
      - Return `{ success: true/false, deliveryId }`

   b. `deliveryLog` query:
      - Input: `z.object({ reportId: z.string() })`
      - Return `prisma.reportDelivery.findMany({ where: { reportId: input.reportId }, orderBy: { createdAt: 'desc' }, take: 50 })`
